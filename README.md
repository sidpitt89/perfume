## Installation

#### 1. Install Node.JS
#### 2. Install NJS module dependencies.
```bash
npm install
```
#### 3. Install client-side browser Javascript dependencies.
```bash
npm install -g bower
bower install
```
#### 4. Inject Bower dependencies.
```bash
gulp wiredep
```
#### 5. Build
```bash
gulp build
```
#### 6. Serve
```bash
gulp serve:dist
```
